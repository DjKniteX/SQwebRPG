# Getting Started

1. Install dependencies with `npm install`.
2. Copy `.env.example` to `.env`.
3. Generate Prisma with `npm run prisma:generate`.
4. Create and seed the database with `npm run setup:demo`.
5. Start the app with `npm run dev`.

Visit `/`, `/demo`, `/login`, `/play`, `/admin`, and `/docs`.
