# Admin Editor Guide

The admin dashboard is available at `/admin`.

It uses a database-editor layout:

- Left sidebar: content categories
- Middle list: entries in that category
- Right editor: JSON editor for the selected entry

Current editable categories include:

- Classes
- Rooms
- Monsters
- NPCs
- Recruitable companions
- Quests
- Items
- Spells
- Dungeons
- Announcements
- Users
- Characters

The editor supports create, save, duplicate, delete, and search. Future versions can add friendlier structured forms on top of the same API.
